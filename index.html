Welcome to Zenith Academy AI

Zenith Academy AI is a responsive mental well-being companion designed specifically for students facing competitive examination stress.

Please note that Demo Mode utilizes local heuristic inference, while unlocking the API Key enables real-time Gemini 2.5 Flash analysis.

Evaluators can choose how they wish to explore the application:

  • Rich Demo Mode: Explore pre-populated analytics, interactive SVG trigger matrix nodes, and keyword-responsive coping chat playgrounds using local heuristic inference.
  • Live Gemini Mode: Connect directly to Google AI Studio's Gemini 2.5 Flash endpoint for real-time journal text analysis and empathetic dialog context.

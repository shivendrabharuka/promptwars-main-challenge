# Zenith Academy AI - Student Mental Well-Being Companion

### Chosen Vertical
* Healthcare & Mental Productivity Automation

### Approach and Logic
* **Context Gathering Layer:** The application dynamically captures individual student preparation parameters (e.g., specific exams like JEE/UPSC, current stress levels).
* **Architectural Stress Matrix:** Uses structured JSON analysis from Gemini 2.5 Flash to automatically classify unstructured journal logs into explicit anxiety vectors (backlog panic, mock test drop stress, peer comparison).
* **Empathetic Companion Engine:** Injects real-time personalized coping mechanisms and interactive grounding tools directly tailored to the identified exam trigger.

### Gen AI Services Utilized
* **Google AI Studio (Gemini 2.5 Flash Endpoint):** Used for advanced JSON data structure classification and interactive empathetic conversational grounding.
* **Google Antigravity IDE:** Workspace file structure isolation and automated tool tracking.

### Assumptions Made
* External model inference latency is baseline managed through optimized browser client-side caching.
